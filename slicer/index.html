Segmentation as code.

A simple JSON spec defines how users qualify.
Evaluate it on your warehouse. Activate it anywhere.

{
  "name": "High Value Repeat Buyers",
  "specification": {
    "inclusion": {
      "relation": "intersection",
      "slices": [
        {
          "type": "past_action",
          "past_action": {
            "user_actions": [
              {
                "event_id": "purchase",
                "occurrence": {
                  "operator": ">=",
                  "count": 2
                }
              }
            ]
          }
        }
      ]
    }
  }
}
